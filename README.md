# Klasterisasi Oligohydramnion Menggunakan Fuzzy C-Means (FCM)

Proyek ini bertujuan untuk melakukan klasterisasi data pasien oligohydramnion menggunakan algoritma Fuzzy C-Means.

## Struktur Proyek
- `data/`: Berisi dataset pasien.
- `outputs/`: Hasil iterasi dan visualisasi.
- `scripts/`: Skrip pemrosesan data.
- `*.ipynb`: Notebook untuk preprocessing, training, testing, dan evaluasi.
